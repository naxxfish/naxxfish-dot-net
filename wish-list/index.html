Wish List

2019-03-10 :: naxxfish

For a long time, I’ve used Amazon Wishlist to help people figure out what sort of things I’d like to be given as a present. I’ve decided that isn’t a particularly great solution, as it doesn’t give people the choice of retailer – especially where there is a local one they could support instead.

So I’ve made my own Wishlist containing a bunch of things that I’d appreciate, which is hosted by GitHub pages, using some code by ndarville to render a nice wishlist template using Jekyll.

I’ve included several suppliers for as many of the items as possible, including local retailers.
